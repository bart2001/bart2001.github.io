Ngrok

Posted at — Oct 14, 2019

서버개발을 할 때 테스팅을 위해서 내 로컬환경을 외부로 열어주어야 하는 경우가 있다. 집에서 개발하는 경우에는 공유기를 통해서 포트포워딩을 해서 열어주어야하나 싶기도 하고 더불어 https 통신을 하기 위해서 SSL 인증서를 발급받는 것도 번거롭다.

ngrok은 외부에서 로컬에 접속 가능하게 만들어 주는 터널링 프로그램이다. 간단하게 말하면 ngrok을 통해서 생성된 url 주소를 통해서 로컬에서 개발중인 서버에 접근할 수 있게 해준다.

ngrok은 다음과 같이 사용이 가능하다.

brew cask install ngrok
ngrok http 8080 #ngrok 생성 URL -> 로컬 8080
